# YOURLS日本語訳
このフォーク及び日本語訳ファイルは、私(torichan)がより自然かつわかりやすい日本語訳へと編集したものです。

実際の使用方法は下記またはフォーク元を参考にしてください。

## インストール
詳細は省きます。下記方法でできるはずです。

1.`ja_JP.mo`ファイルをダウンロード   

2.`config.php` の下記行を編集します。(見つからない場合は追加します。)    
- `define( 'YOURLS_LANG', 'ja_JP' );`        

3.`user/languages`フォルダへと移動し、ダウンロードした`ja_JP.mo`ファイルをアップロードします。

## 翻訳ファイルを編集
日本語訳に違和感を感じたら翻訳を編集できます。  

1.`ja_JP.po`ファイルをダウンロードしメモ帳など任意のソフトウェアで編集します。  

2.`msgfmt -o ja_JP.mo ja_JP.po`とコマンドを利用するか、オンラインコンバーターを利用すると良いでしょう。(po to mo)  

3.`user/languages`へ、変換した`ja_JP.mo`ファイルをアップロードします。
